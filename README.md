# Dengine
Dengine é uma "Engine" feita em lua(Ela não é literalmente uma Engine, como Unity, é apenas um pacote para facilitar a criação de programas).

# As funcionalidades dela são:

Draw: É uma função para exibir "imagens" em txt.

Timer: É, como diz o nome, um timer, cujo pode-se escolher quando tempo terá de espera.

Frame: É como um frame de um vídeo, usa-se como o Draw, mas dá também para ajudtar o tempo de duração do frame.

Logo: Exibe na tela o logo da Dengine.

Crash: Exibe uma mensagem várias vezes(pode crashar seu programa)

# OBS: 
Esse programa pode não funcionar em seu dispositivo, mas você poderá mudar o código para que funcione corretamente.
